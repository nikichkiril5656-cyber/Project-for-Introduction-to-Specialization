=== Квантовый Кликер ===

ИНСТРУКЦИЯ ПО УСТАНОВКЕ:
1. Распакуйте архив в любую папку
2. Запустите игру двойным кликом по start_game.bat
3. Или запустите напрямую game.exe

УПРАВЛЕНИЕ В ИГРЕ:
- ЛКМ: Кликать по кнопке
- Пробел: Быстрый клик
- F11: Переключение полноэкранного режима
- ESC: Выход из игры
- Ctrl+S: Быстрое сохранение
- Ctrl+L: Быстрая загрузка

Сохранения автоматически создаются в папке "saves"
